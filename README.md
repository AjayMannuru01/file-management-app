File Management App
A simple command-line file management application in Python that allows users to create, view, read, edit, and delete text files from their working directory.

Features
Create a file: Add new files safely, preventing overwriting of existing files

View all files: List all files in the current directory

Read a file: Display the content of a text file

Edit a file: Add new content to an existing file

Delete a file: Remove files from the directory

Requirements
Python 3.x

No external dependencies (all modules from Python standard library)
